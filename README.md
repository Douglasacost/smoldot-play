# smoldot-play

## How to run?
- run `yarn intall`
- then `yarn start` to run the index.ts

## Arguments

> `yarn start [network] [maxBatchSize] [batchInterval]`

- network: `testnet` | `mainnet` - default: `testnet`
- maxBatchSize: (int) - default: `1`
- batchInterval: (ms) - default: `1000`
